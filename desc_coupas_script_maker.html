프로그램 소개

이 프로그램은 유튜브 쿠팡 제휴 마케팅을 진행하시는 모든 분들이 쿠팡 제품 링크만 넣으면 자동으로 제품 정보를 추출하고, 자동으로 AI가 사용자가 정의한 프롬프트를 참고하여 그에 맞게 대본 스크립트를 생성해주는 완전 자동화된 솔루션입니다.

주요 기능

  • 쿠팡 제품 링크 입력 시 썸넬 및 상세페이지 자동 저장
  • 제품 정보 자동 크롤링 및 추출
  • 한 번에 n개의 링크를 입력하여 n개의 묶음 추출
  • 묶음 생성 ON/OFF 기능
    • ON: 여러 제품을 하나의 세트로 묶어 비교/추천하는 통합 스크립트 생성
    • OFF: 각 제품별 개별 스크립트 생성
  • AI 제목 및 대본 동시 생성 기능
  • 사용자 정의 프롬프트 설정 기능
  • 에이전트 역할 커스터마이징 기능

작동 방식

  • 쿠팡 제품 링크 입력
  • 썸넬 및 상세페이지 자동 저장
  • 제품 정보 자동 크롤링 및 추출
  • 사용자가 원하는 프롬프트 및 에이전트 역할 설정
  • AI가 설정된 프롬프트에 따라 제목 생성
  • AI가 설정된 프롬프트에 따라 대본 생성
